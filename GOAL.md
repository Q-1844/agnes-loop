# Goal: Build a Simple Landing Page

Create a responsive landing page with the following features:

## Requirements

1. **Header** with navigation links
2. **Hero section** with a headline and call-to-action button
3. **Features section** showing 3 feature cards
4. **Footer** with contact information
5. **Responsive design** — works on mobile and desktop
6. **Dark mode toggle** — switch between light and dark themes

## Tech Stack

- HTML5 + CSS3 + Vanilla JavaScript
- No frameworks (pure HTML/CSS/JS)
- Mobile-first responsive design

## Success Criteria

- Page loads without errors
- All sections are visible
- Responsive layout works at 320px, 768px, and 1024px widths
- Dark mode toggle works
- No console errors
